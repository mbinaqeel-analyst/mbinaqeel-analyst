🛒 Maven Market: End-to-End Business Intelligence Pipeline
A comprehensive Data Engineering and Analytics project transforming raw retail data into actionable business intelligence. This project demonstrates a full-stack analytical workflow—from SQL database modeling to Python-driven performance reporting.

🛠️ Project Overview
This analysis provides a deep dive into the Maven Market dataset, focusing on revenue trends, customer segmentation, and operational efficiency. By leveraging a Star Schema architecture, this project ensures high-performance querying and accurate KPI tracking.

🚀 Key Technical Highlights
ETL Pipeline: Automated ingestion of multi-year transaction data into a normalized SQLite database using Python and SQLAlchemy.

Data Modeling: Implemented a Relational Star Schema (Fact & Dimension tables) to maintain data integrity and optimize analytical speed.

Feature Engineering: Calculated 15+ complex business KPIs, including Year-over-Year (YoY) growth, Churn Rates, and Inventory Fulfillment Latency.

Automated Reporting: Developed a custom Python script to generate a 9-page Executive PDF Report with a professional corporate color palette.

📈 Strategic Insights
Retention: Identified a significant churn rate, providing a clear basis for re-engagement strategies.

Efficiency: Analyzed order fulfillment bottlenecks by mapping regional return volumes against processing times.

Segmentation: Pinpointed "Professional" and "Management" occupation tiers as the primary drivers of high-value revenue.
