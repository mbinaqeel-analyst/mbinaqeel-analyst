# Customer Churn Analysis

## Problem Statement
Analyze customer data to identify patterns and factors contributing to customer churn.

## Dataset
Customer-level dataset containing demographics, usage metrics, and churn labels.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- SQL (for data extraction and aggregation)

## Key Insights
- Higher churn observed among customers with shorter tenure
- Customers with higher monthly charges showed increased churn risk
- Certain service types were strongly associated with churn

## Outcome
The analysis highlights key churn drivers that can help businesses design
targeted retention strategies.
