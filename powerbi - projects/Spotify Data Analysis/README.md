🎵 Spotify 360° Business Intelligence Suite
Solving Data Fragmentation with Microsoft Fabric & Power BI

📖 Project Summary
This project takes raw, siloed Spotify data (Artists, Tracks, and Audio Features) and combines them into one interactive dashboard. It allows music executives to see the "Formula for Success" by connecting an artist's popularity to the technical vibe of their music.

🛠️ Tech Stack
Tool: Power BI Desktop / Microsoft Fabric

Data Source: Spotify Web API (CSV)

Modeling: Star Schema (1:Many relationships)

Language: DAX (Data Analysis Expressions)

📊 The 4-Page Dashboard
Overview: The "Big Picture" metrics like Total Tracks and Average Popularity.

Artist Performance: A deep dive into which artists are consistent hit-makers.

Audio Anatomy: A technical look at how Tempo, Energy, and Danceability drive hits.

Strategy Insights: A text-based summary for executives to read the "bottom line" quickly.

💡 Key Business Insights
The Tempo "Sweet Spot": Songs between 115-125 BPM are currently 15% more popular.

Explicit Content: Explicit tracks peak higher in popularity but have fewer playlist placements than clean versions.

Duration Trends: The most successful songs in 2023-2025 are getting shorter, averaging 2:45 to 3:15 minutes.
